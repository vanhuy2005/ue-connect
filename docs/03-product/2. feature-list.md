# Feature List

## 1. Purpose

File này liệt kê toàn bộ feature của UEConnect theo **thứ tự nên dev**, đồng thời mapping từng feature về file `feature-specs`.

Mục tiêu:

- Biến product overview thành danh sách feature rõ ràng.
- Chia feature theo phase dev.
- Map feature với priority, dependency, requirement, page/spec và test direction.
- Giúp dev không build theo cảm hứng lúc nửa đêm, một hoạt động nghe sáng tạo nhưng thường tạo ra database schema khiến đời sau nguyền rủa.
- Làm nền cho `feature-priority.md`, `sitemap.md`, feature specs, backlog, sprint planning và test planning.

---

## 2. Development Order Philosophy

UEConnect không dev theo thứ tự “feature nào nhìn vui thì làm trước”.

Thứ tự dev phải đi theo nguyên tắc:

```txt
01. Foundation trước feature
02. Identity trước social
03. Permission trước admin action
04. Profile trước discovery
05. Greeting trước messaging
06. Report/block trước mở rộng community
07. Moderation trước scale content
08. Audit trước admin power
09. PWA shell trước polish
10. Analytics sau khi core events có chỗ phát sinh
````

Nói gọn:

```txt
Không có verified identity thì không có trust.
Không có permission thì không có governance.
Không có safety thì không nên mở social.
Không có audit thì admin power chỉ là một tai nạn đang chờ lịch diễn.
```

# 10. Feature Risk Map

| Feature Area    | Main Risk                      | Mitigation                                    |
| --------------- | ------------------------------ | --------------------------------------------- |
| Verification    | Fake identity, duplicate MSSV  | Evidence, admin review, uniqueness check      |
| Evidence upload | Sensitive data leak            | Private storage, permission viewer            |
| Profile         | MSSV leak, dating-like profile | Hide MSSV, context-first fields               |
| Feed            | Spam/unsafe content            | Report, moderation, rate limit later          |
| Discovery       | Dating vibe                    | No dating terms, shared context, Greeting CTA |
| Greeting        | Spam greetings                 | Duplicate prevention, rate limit              |
| Messaging       | Random DM/spam                 | Permission-based access                       |
